## 等价类划分

<table>
	<tr>
		<th>输入条件</th>
		<th>有效等价类</th>
		<th>无效等价类</th>
	</tr>
	
	<tr>
		<td>词库选择</td>
		<td>(P1) n, v, adv... 中的选项</td>
		<td>(N1) 不选择词库</td>
	</tr>

    <tr>
        <td>起始单词选择</td>
        <td>
            <ul>
                <li>(P2) 输入单词</li>
                <li>(P3) 选择第一个单词开始</li>
                <li>(P4) 选择从上次结束单词开始</li>
            </ul>
        </td>
        <td>(N2) 不选择</td>
    </tr>
    
	<tr>
        <td>(选择起始单词时) 单词输入</td>
        <td>(P5) 与显示备选单词相同的词</td>
        <td>
            <ul>
                <li>(N3) 与显示备选单词不同的文本</li>
                <li>(N4) 不输入</li>
                <li>(N5) 空白符</li>
            </ul>
        </td>
    </tr>

    <tr>
        <td>背单词数量</td>
        <td>(P6) 合法数量</td>
        <td>
            <ul>
                <li>(N6) 超出最大值的数</li>
                <li>(N7) 零</li>
                <li>(N8) 负数</li>
                <li>(N12)浮点数</li>
                <li>(N13)非数字字符</li>
            </ul>
        </td>
    </tr>

    <tr>
        <td>(背单词时) 英文单词输入</td>
        <td>(P7) 正确输入</td>
        <td>
            <ul>
                <li>(N9) 不正确的文本</li>
                <li>(N10) 不输入</li>
                <li>(N11) 空白符</li>
            </ul>
        </td>
    </tr>
</table>

**Attention**:

P is for Positive (有效). N is for Negative (无效).
